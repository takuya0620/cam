# camera.py
macのデフォルトカメラを用いて画像を連続して取得することで動画を取得する. 
# スクリプト
次のスクリプトで動画をウィンドウ表示した. ウィンドウを終了する際にESCキーを用いた. 

    >>> import camera
    >>> camera.capture_camera()
    >>> exit()
